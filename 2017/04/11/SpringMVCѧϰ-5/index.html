SpringMVC-拦截器

发表于 2017-04-11 | 分类于 SpringMVC | | 次阅读
字数统计: 673 | 阅读时长 ≈ 3

​ SpringMVC 中的 Interceptor 拦截器也是相当重要和相当有用的，它的主要作用是拦截用户的请求并进行相应的处理。比如通过它来进行权限验证，或者是来判断用户是否登陆等操作。

对于 springmvc 拦截器的定义方式有两种方式

  1. 实现接口： org.springframework.web.servlet.HandlerInterceptor

  2. 继承适配器

    org.springframework.web.servlet.handler.HandlerInterceptorAdapter
