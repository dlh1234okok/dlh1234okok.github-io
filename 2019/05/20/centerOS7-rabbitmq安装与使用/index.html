centerOS7-rabbitmq安装与使用

发表于 2019-05-20 | 分类于 RabbitMq | | 次阅读
字数统计: 3k | 阅读时长 ≈ 18

erlang环境安装

下载http://erlang.org/download/otp_src_19.3.tar.gz

解压缩 tar -zvxf

进入到解压缩后的源码目录

./configure –prefix=/opt/tools/erlang –without-javac –with-ssl=openssl目录

安装需要的环境

gcc gcc++

ncurses-devel

openssl

…

在Configure的时候会出现错误提示，看到什么依赖没有就安装什么，建议以tar包形式安装，不要使用rpm的方式，rpm安装后总是没有反应，还是没有添加上依赖

然后

make && make install

RabbitMQ下载安装

下载地址http://www.rabbitmq.com/releases/rabbitmq-server/v3.6.1/rabbitmq-server-generic-unix-3.6.1.tar.xz

解压

xz -d rabbitmq-server-generic-unix-3.6.1.tar.xz

tar -zvxf rabbitmq-server-generic-unix-3.6.1.tar

在解压后的目录/rabbitmq_server-3.6.1/etc/rabbitmq下，添加配置文件rabbitmq.config && rabbitmq-env.config

配置erlang 和 RabbitMQ的环境变量

启动RabbitMQ rabbitmq-server start

在安装时出现最大的问题就是erlang和rabbitmq版本不匹配导致无法安装启动成功，记录下我安装成功的版本为了以后方便使用，期间还有另外各种问题不一一记录了，各位见招拆招。
